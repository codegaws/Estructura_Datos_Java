# Iconos para Colecciones en Markdown 📚

A continuación, algunos iconos útiles para documentar colecciones en Java:

- **Colecciones generales**: 📚
- **Listas (List)**: 🗂️
- **Conjuntos (Set)**: 🧺
- **Colas (Queue)**: 📦
- **Mapas (Map)**: 🗺️
- **Agregar elemento**: ➕
- **Eliminar elemento**: ➖
- **Buscar elemento**: 🔍
- **Clave/Valor**: 🏷️

Ejemplo de uso:

```markdown
# Colecciones en Java 📚

## Tipos principales
- Listas 🗂️
- Conjuntos 🧺
- Colas 📦
- Mapas 🗺️

## Operaciones comunes
- Agregar ➕
- Eliminar ➖
- Buscar 🔍
- Clave/Valor 🏷️
```
